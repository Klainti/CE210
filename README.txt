
   README (read before trying the code)
------------------------------------------------

1. Rename __build.xml to build.xml (mandatory step).
2. To compile and build jar file using ANT tool (https://ant.apache.org) run the command
   $> ant all
3. To clean everything and rebuild run the command
   $> ant clean
   
ANT for Linux is on any Linux distribution (Ubuntu, Suse etc). For example in Ubuntu Linux the package is the following
    ant - Java based build tool like make 
    
ANT for Windows can be obtained by downloading and executing WINANT (https://code.google.com/p/winant/)

Happy coding!
